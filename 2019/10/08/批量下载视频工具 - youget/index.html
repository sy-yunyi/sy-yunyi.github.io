批量下载视频工具 - youget

发表于 2019-10-08 | 分类于 SIX不言 | |
字数统计: 1.3k | 阅读时长 ≈ 6

一个可以在命令行批量下载网页资源的小工具。项目地址：https://github.com/soimort/you-get

中文介绍地址：https://github.com/soimort/you-get/wiki/%E4%B8%AD%E6%96%87%E8%AF%B4%E6%98%8E

安装要求

  • Python 3
  • FFmpeg(强烈推荐) or Libav
  • (可选) RTMPDump

使用简单方便：

安装

安装方法，官方提供了6种安装方式，这里只介绍两种最方便的方式

  1. 使用pip安装

  2. 使用安装包（Windows）

    exe (单独文件) 或 7z (包括所有依赖) 可从https://github.com/soimort/you-get/releases/latest 下载

    下载解压后，找到setup.py，运行

简单使用教程

  1. 查看网页视频信息--info/-i

    标有DEFAULT 为默认画质。如果想更换格式，可以使用--itag 选项。下载mp4，设置--itag=18。

  2. 下载视频

  3. 指定下载输出目录-o/--output-dir

  4. 指定输出文件名-O/--output-filename

  5. 批量下载-l/--playlist

  6. 暂停下载

    使用Ctrl+C暂停下载，下次使用you-get传入相同参数时，下载将从上次继续开始【断点续传】。

  7. 观看视频-p/--player

  8. 加载cookie，--cookies/-c

下载其他内容

you-get不仅可以下载视频，还可以下载网页上的图片等其他内容。

如果只是网页链接，you-get会自动检查网页，下载一切感兴趣的东西。
