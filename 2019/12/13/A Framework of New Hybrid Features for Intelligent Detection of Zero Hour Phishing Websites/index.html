笔记|A Framework of New Hybrid Features for Intelligent Detection of Zero Hour Phishing Websites

Six's Blog
2019-12-13
论文笔记
机器学习, 钓鱼检测

来源

Nagunwa T, Naqvi S, Fouad S, et al. A Framework of New Hybrid Features for Intelligent Detection of Zero Hour Phishing Websites[C]//International Joint Conference: 12th International Conference on Computational Intelligence in Security for Information Systems (CISIS 2019) and 10th International Conference on EUropean Transnational Education (ICEUTE 2019). Springer, Cham, 2019: 36-46.

介绍

针对现有的钓鱼检测方法，钓鱼攻击者学习其特征，采用精心设计的方法来绕过检测。本文提出了一个通过引入混合特征的零时（Zero Hour）钓鱼站点检测框架。其指出域名注册信息和页面信誉值在检测过程中占有非常重要的地位。每个页面的预测时间为7.63s 。应用了5种类型31个特征，并且对8种机器学习算法进行了测试。

Zero hour 未知的新的钓鱼站点

根据Webroot的最新数据，平均每月有140万个独特的钓鱼网站被创建——每天有46000个新的钓鱼网站被创建——它们平均只活跃4-8个小时。https://www.webroot.com/us/en/business/threat-intelligence/internet/real-time-anti-phishing

相关工作

  • Machine learning based phishing detection from URLs [2019]：提取了1701个单词和40个基于自然语言的特征，应用随机森林算法，准确率97.98
  • New hybrid features for phish website prediction [2016]：设计了58个预测特征，包括页面结构和URL相关的特征，应用SVM算法，FP1.17，FN0.03
  • A stacking model using URL and HTML features for phishing webpage detection [2019]：设计了20个特征，应用XGBoost和LightGBM算法，准确率97.3，错误率4.46，FN1.61
  • Towards detection of phishing websites on client-side using machine learning based approach [2018]：与第三个相比没有采用第三方的特征，
  • Predicting phishing websites based on selfstructuring neural network[2014]：采用了三种混合特征，5个页面结构特征，9个URL信息特征和3个域名注册信息特征，应用深度神经网络，准确率92.18
  • A comprehensive and efficacious architecture for detecting phishing webpages [2014]：采用了页面结构、URL、域名注册和页面信誉值相关的30个特征，应用深度神经网络，正确率97.71

框架设计

  • 个人信息页面过滤：过滤不包含个人信息的页面，这种页面不是钓鱼页面。
  • 钓鱼黑名单检查
  • URL重定向检查
    • 客户端重定向
    • 服务端重定向
    • 短链接
  • 特征提取
  • 训练分类器
  • 预测分析

预测特征

  • 页面结构和内容
  • URL结构
  • 域名注册信息
  • SSL证书信息
  • 页面信誉度

实验

从实验结果来看，随机森林的效果最好。

上图为每组特征单独训练随机森林的正确率，可以看出，仅仅使用域名注册信息特征就可以达到99的正确率。但是从上面的每个特征重要的程度来看，域名注册信息的重要程度并没有页面信誉度的重要程度高，这里猜测可能和数据集相关。
